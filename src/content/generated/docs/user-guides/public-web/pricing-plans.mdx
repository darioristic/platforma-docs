# Cloud Factory - Pricing Plans

Cloud Factory supports flexible, operator-defined pricing. Each telecom operator configures their own product catalog, pricing tiers, and billing rules through the admin platform. The examples below illustrate the standard tier structure.

## Enterprise VPS

| Tier       | vCPU | RAM   | SSD Storage | Bandwidth | Uptime SLA |
| ---------- | ---- | ----- | ----------- | --------- | ---------- |
| Basic      | 2    | 4 GB  | 80 GB       | 2 TB      | 99.5%      |
| Business   | 4    | 8 GB  | 160 GB      | 5 TB      | 99.9%      |
| Enterprise | 8    | 32 GB | 500 GB      | Unlimited | 99.95%     |

All VPS tiers include DDoS protection, automated daily backups, and root SSH access.

## Private Cloud

Custom configurations tailored to enterprise workloads. Resources are allocated from dedicated pools with guaranteed isolation. Typical deployments include:

- Custom vCPU, RAM, and storage combinations
- Private networking with VLAN support
- Dedicated hypervisors (optional)
- Custom SLA agreements up to 99.99%

Pricing is quote-based and configured per tenant in the admin platform.

## Managed Kubernetes

Production-ready clusters with managed control planes, auto-scaling node pools, and integrated monitoring. Billed per worker node with control plane included.

## AI/ML & Data

GPU-accelerated instances for training and inference workloads. Includes access to pre-configured ML frameworks and data pipeline tooling. Billed per GPU-hour or as reserved instances.

## Managed Databases

| Engine     | Replication        | Automated Backups    | Point-in-Time Recovery |
| ---------- | ------------------ | -------------------- | ---------------------- |
| PostgreSQL | Primary-replica    | Daily + WAL archival | Yes                    |
| MySQL      | Primary-replica    | Daily                | Yes                    |
| MongoDB    | Replica set        | Daily snapshots      | Yes                    |
| Redis      | Sentinel / Cluster | RDB + AOF            | No                     |

Billed by instance size (vCPU + RAM) and storage consumed.

## Object Storage

S3-compatible object storage with configurable redundancy. Billed per GB stored and per 10,000 API requests. Free ingress; egress billed per GB.

## Add-Ons

| Add-On              | Description                                                  |
| ------------------- | ------------------------------------------------------------ |
| Advanced Monitoring | Grafana dashboards, custom alerts, 90-day metric retention   |
| Backup Plus         | Hourly snapshots, cross-region replication, 1-year retention |
| DDoS Protection Pro | Layer 7 filtering, traffic scrubbing, real-time mitigation   |
| Managed Firewall    | Custom rule sets, intrusion detection, audit logging         |
| Load Balancer       | L4/L7 balancing with health checks and SSL termination       |
| Private Networking  | Isolated VLANs, VPN gateways, peering                        |

> **Note**: All pricing shown is illustrative. Actual rates are configured per operator deployment through the Cloud Factory admin platform product catalog.
