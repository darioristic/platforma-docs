# Billing Integration Service

**Port:** 3005 | **Database:** PostgreSQL | **Events:** Kafka (consumer + producer) | **Payments:** Stripe

Manages the complete billing cycle: subscriptions, invoices, payments, usage tracking, rating engine, refunds, and Stripe integration.

---
