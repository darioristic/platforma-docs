# Identity Service

**Port:** 3003 | **Database:** PostgreSQL | **Events:** Kafka (producer) | **Auth:** JWT + OAuth2 + MFA

Manages authentication, authorization, users, tenants, roles, API keys, and audit logs. Supports OAuth2 (Google, GitHub, Microsoft) and MFA.

---

| Module                   | Responsibility                                                         |
| ------------------------ | ---------------------------------------------------------------------- |

## 5. Security

### JWT Payload

```json
{
  "sub": "user-id",
  "email": "user@example.com",
  "tenantId": "tenant-id",
  "roles": ["admin"],
  "mfaVerified": true,
  "type": "access",
  "iat": 1234567890,
  "exp": 1234568790
}
```

### Global Guards

| Guard            | Description                                |
| ---------------- | ------------------------------------------ |
| `JwtAuthGuard`   | Validates JWT on all routes except @Public |
| `RolesGuard`     | Checks @Roles decorator                    |
| `TenantGuard`    | Enforces tenant isolation                  |
| `ThrottlerGuard` | Rate limiting (100 req/60s default)        |

### OAuth2 Providers

- **Google** (passport-google-oauth20) — scope: email, profile
- **GitHub** (passport-github2)
- **Microsoft** (passport-microsoft)

All conditionally registered based on env variables (CLIENT_ID + CLIENT_SECRET).
