## Common Characteristics

- **Auth:** JWT + MFA + OAuth2 (Google, GitHub, Microsoft)
- **Multi-tenancy:** Tenant isolation across all services
- **Events:** Kafka (30+ topics, saga pattern, DLQ)
- **Resilience:** Circuit breaker, bulkhead, timeout
- **Caching:** Redis 7
- **Health:** Liveness (`/health`) + Readiness (`/ready`) probes
- **Observability:** Correlation ID propagation, Prometheus metrics
- **Security:** Service-to-service auth, rate limiting, idempotency
