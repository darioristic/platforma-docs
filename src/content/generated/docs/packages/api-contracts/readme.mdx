# @cloud-factory/api-contracts

Shared TypeScript type definitions and interfaces for all Cloud Factory services and frontends.

## Installation

```json
"@cloud-factory/api-contracts": "*"
```

## Key Exports

| Module            | Contents                                                   |
| ----------------- | ---------------------------------------------------------- |
| `product`         | `Product`, `ProductCategory`, `CreateProductRequest`, etc. |
| `order`           | Order DTOs, statuses, request/response types               |
| `identity`        | User, tenant, auth-related types                           |
| `billing`         | Invoice, payment, subscription types                       |
| `provisioning`    | Resource types, provisioning status, provider contracts    |
| `pricing-plan`    | Pricing plan definitions (deprecated in favor of products) |
| `events`          | `VersionedEvent` base interface for Kafka event payloads   |
| `admin-analytics` | Admin dashboard analytics DTOs                             |
| `correlation`     | Request correlation/tracing types                          |
| `crm`             | CRM-related types for admin platform                       |

A versioned namespace `v1` is also available for explicit version pinning.

## Usage

```typescript
import { Product, ProductCategory, VersionedEvent } from '@cloud-factory/api-contracts';

// Or use versioned namespace
import { v1 } from '@cloud-factory/api-contracts';
const product: v1.Product = { ... };
```
