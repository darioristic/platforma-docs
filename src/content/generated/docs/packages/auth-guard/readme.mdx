# @cloud-factory/auth-guard

NestJS module providing JWT authentication, role-based access control, tenant isolation, and service-to-service auth guards.

## Installation

```json
"@cloud-factory/auth-guard": "*"
```

## Key Exports

- **`CloudFactoryAuthModule`** -- Global dynamic module; registers all guards automatically via `APP_GUARD`.
- **Guards:** `JwtAuthGuard`, `RolesGuard`, `TenantGuard`, `ServiceAuthGuard`
- **Decorators:** `@CurrentUser()`, `@Public()`, `@Roles(...)`, `@ServiceOnly()`, `@EffectiveTenantId()`
- **Types:** `JwtPayload`, `CurrentUserPayload`, `AuthModuleOptions`
- **Utilities:** `isPlatformAdmin()`, `resolveTenantId()`

## Usage

### Register in AppModule

```typescript
import { CloudFactoryAuthModule } from '@cloud-factory/auth-guard';
import { ConfigService } from '@nestjs/config';

@Module({
  imports: [
    CloudFactoryAuthModule.forRootAsync({
      useFactory: (config: ConfigService) => ({
        jwtSecret: config.get('JWT_SECRET'),
      }),
      inject: [ConfigService],
    }),
  ],
})
export class AppModule {}
```

### Protect endpoints

```typescript
import { CurrentUser, Public, Roles, ServiceOnly } from '@cloud-factory/auth-guard';

@Controller('orders')
export class OrderController {
  @Public()
  @Get('health')
  health() {
    return 'ok';
  }

  @Roles('admin')
  @Get()
  list(@CurrentUser() user: CurrentUserPayload) {
    // user.tenantId, user.roles, user.email available
  }

  @ServiceOnly()
  @Post('internal')
  internalOnly() {
    /* only service tokens accepted */
  }
}
```
