# @cloud-factory/resilience

Circuit breaker, bulkhead, retry, and timeout patterns for NestJS services. Includes Prometheus metrics and Terminus health checks.

## Installation

```json
"@cloud-factory/resilience": "*"
```

## Key Exports

| Export                           | Description                                             |
| -------------------------------- | ------------------------------------------------------- |
| `ResilienceModule`               | Global module registering all resilience services       |
| `@CircuitBreaker(name, config?)` | Method decorator -- wraps calls with circuit breaker    |
| `CircuitBreakerService`          | Programmatic circuit breaker (`execute()`)              |
| `@Bulkhead(name, config?)`       | Method decorator -- limits concurrent executions        |
| `BulkheadService`                | Programmatic bulkhead with queue                        |
| `@Timeout(ms)`                   | Method decorator -- throws `TimeoutError` on expiry     |
| `RetryInterceptor` / `retry()`   | NestJS interceptor for automatic retries                |
| `ResilienceMetrics`              | Prometheus counters for breaker trips, bulkhead rejects |
| `ResilienceHealthIndicator`      | Reports circuit states to Terminus health checks        |

## Usage

### Register the module

```typescript
import { ResilienceModule } from '@cloud-factory/resilience';

@Module({
  imports: [ResilienceModule.forRoot()],
})
export class AppModule {}
```

### Decorate methods

```typescript
import { CircuitBreaker, Bulkhead, Timeout } from '@cloud-factory/resilience';

@Injectable()
export class PaymentGateway {
  @CircuitBreaker('stripe', { failureThreshold: 5, resetTimeoutMs: 30000 })
  @Timeout(5000)
  async charge(amount: number): Promise<Receipt> {
    return this.stripeClient.charge(amount);
  }

  @Bulkhead('email-send', { maxConcurrent: 10, maxQueue: 50 })
  async sendReceipt(receipt: Receipt): Promise<void> {
    // limited to 10 concurrent calls, 50 queued
  }
}
```

### Retry interceptor

```typescript
@UseInterceptors(new RetryInterceptor({ maxRetries: 3, delayMs: 1000 }))
@Get('external-data')
async fetchData() { /* ... */ }
```
