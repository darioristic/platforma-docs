# @cloud-factory/common

Shared NestJS infrastructure: service bootstrap, domain errors, logging, metrics, throttling, and database utilities.

## Installation

```json
"@cloud-factory/common": "*"
```

## Key Exports

| Export                                           | Description                                                                          |
| ------------------------------------------------ | ------------------------------------------------------------------------------------ |
| `ServiceCoreModule`                              | One-liner service bootstrap (auth, cache, vault, throttler, resilience, Sentry)      |
| `SharedInfrastructureModule`                     | Correlation-ID middleware, HTTP exception filter, sanitize pipe                      |
| `ServiceHttpModule` / `ServiceTokenService`      | Service-to-service authenticated HTTP calls                                          |
| `CloudFactoryLoggerModule`                       | Pino-based structured logging                                                        |
| `MetricsModule`                                  | Prometheus metrics via `prom-client`                                                 |
| `DomainError` hierarchy                          | Standardized errors: `EntityNotFoundError`, `ValidationError`, `ConflictError`, etc. |
| `DbPoolHealthIndicator` / `DbPoolMetricsService` | TypeORM connection pool monitoring                                                   |
| `createTypeOrmConfig`                            | TypeORM config factory with Vault secret loading                                     |
| `CorrelationIdMiddleware`                        | Propagates `x-correlation-id` across requests                                        |
| `DeprecationMiddleware`                          | Marks API versions as deprecated with response headers                               |
| `SanitizePipe`                                   | Input sanitization pipe                                                              |

## Usage

### Bootstrap a service with all infrastructure

```typescript
import { ServiceCoreModule } from '@cloud-factory/common';

@Module({
  imports: [
    ServiceCoreModule.forRoot({
      cacheTtl: 300,
      throttleLimit: 20,
      serviceName: 'order-service',
    }),
  ],
})
export class AppModule {}
```

### Throw domain errors

```typescript
import { EntityNotFoundError, BusinessRuleViolationError } from '@cloud-factory/common';

throw new EntityNotFoundError('Order', orderId);
throw new BusinessRuleViolationError('MAX_ITEMS', 'Order exceeds max item count');
```

### Service-to-service calls

```typescript
import { ServiceHttpModule } from '@cloud-factory/common';

@Module({
  imports: [ServiceHttpModule],
})
export class OrderModule {}
```
