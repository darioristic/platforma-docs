│  ● Tenant isolation: TenantGuard on all requests                           │
│                                                                             │
│  API Security                                                               │
│  ────────────                                                              │
│  ● Rate limiting: 5/s auth, 2/s register, 30/s general                    │
│  ● API Key authentication (separate from JWT)                              │
