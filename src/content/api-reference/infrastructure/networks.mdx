# Networks

<ApiEndpoint method="GET" path="/v1/infrastructure/networks" auth="Bearer token">
  List all virtual networks.
</ApiEndpoint>

## Query Parameters

<ParamTable>
  <Param name="cluster_id" type="string">
    Filter by cluster.
  </Param>
  <Param name="type" type="string">
    Filter by type: `private`, `public`, `management`.
  </Param>
</ParamTable>
