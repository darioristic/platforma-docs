# Instances

<ApiEndpoint method="GET" path="/v1/infrastructure/instances" auth="Bearer token">
  List all virtual machine instances.
</ApiEndpoint>

## Query Parameters

<ParamTable>
  <Param name="cluster_id" type="string">
    Filter by parent cluster.
  </Param>
  <Param name="status" type="string">
    Filter by status: `running`, `stopped`, `provisioning`, `error`.
  </Param>
</ParamTable>
