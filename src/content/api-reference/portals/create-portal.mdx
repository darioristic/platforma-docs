# Create Portal

<ApiEndpoint method="POST" path="/v1/portals" auth="Bearer token">
  Creates a new portal instance for the authenticated tenant.
</ApiEndpoint>

## Request Body

<ParamTable>
  <Param name="name" type="string" required>
    Display name for the portal.
  </Param>
  <Param name="type" type="string" required>
    Portal type: `customer`, `admin`, or `partner`.
  </Param>
  <Param name="domain" type="string" required>
    Custom domain for the portal (e.g., `portal.example.com`).
  </Param>
  <Param name="branding" type="object">
    Custom branding configuration including logo URL, primary color, and favicon.
  </Param>
  <Param name="features" type="string[]">
    List of enabled features: `self_service`, `billing`, `support`, `monitoring`.
  </Param>
</ParamTable>
