# List Portals

<ApiEndpoint method="GET" path="/v1/portals" auth="Bearer token">
  Retrieves a paginated list of all portals accessible to the authenticated tenant.
</ApiEndpoint>

## Query Parameters

<ParamTable>
  <Param name="limit" type="integer">
    Number of results per page (default: 20, max: 100).
  </Param>
  <Param name="cursor" type="string">
    Pagination cursor from a previous response.
  </Param>
  <Param name="status" type="string">
    Filter by portal status: `active`, `inactive`, `suspended`.
  </Param>
  <Param name="type" type="string">
    Filter by portal type: `customer`, `admin`, `partner`.
  </Param>
</ParamTable>
