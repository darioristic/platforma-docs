# Subscriptions

<ApiEndpoint method="GET" path="/v1/billing/subscriptions" auth="Bearer token">
  List all subscriptions for a tenant.
</ApiEndpoint>

## Query Parameters

<ParamTable>
  <Param name="customer_id" type="string">
    Filter by customer.
  </Param>
  <Param name="status" type="string">
    Filter by status: `active`, `paused`, `cancelled`, `expired`.
  </Param>
</ParamTable>

## Create Subscription

<ApiEndpoint method="POST" path="/v1/billing/subscriptions" auth="Bearer token">
  Create a new recurring subscription.
</ApiEndpoint>

<ParamTable>
  <Param name="customer_id" type="string" required>
    The customer to create the subscription for.
  </Param>
  <Param name="product_id" type="string" required>
    Product from the catalog.
  </Param>
  <Param name="quantity" type="integer" required>
    Number of units.
  </Param>
  <Param name="billing_cycle" type="string" required>
    Billing frequency: `monthly`, `quarterly`, `annual`.
  </Param>
  <Param name="start_date" type="string">
    When the subscription should start (defaults to now).
  </Param>
</ParamTable>
