# Invoices

<ApiEndpoint method="GET" path="/v1/billing/invoices" auth="Bearer token">
  List all invoices with optional filtering by customer, status, and date range.
</ApiEndpoint>

## Query Parameters

<ParamTable>
  <Param name="customer_id" type="string">
    Filter invoices by customer.
  </Param>
  <Param name="status" type="string">
    Filter by status: `draft`, `issued`, `paid`, `overdue`, `void`.
  </Param>
  <Param name="from" type="string">
    Start date (ISO 8601 format).
  </Param>
  <Param name="to" type="string">
    End date (ISO 8601 format).
  </Param>
</ParamTable>
