# List Orders

<ApiEndpoint method="GET" path="/v1/orders" auth="Bearer token">
  Retrieves a paginated list of all orders.
</ApiEndpoint>

## Query Parameters

<ParamTable>
  <Param name="limit" type="integer">
    Number of results per page (default: 20, max: 100).
  </Param>
  <Param name="cursor" type="string">
    Pagination cursor from a previous response.
  </Param>
  <Param name="status" type="string">
    Filter by status: `pending`, `processing`, `provisioned`, `failed`, `cancelled`.
  </Param>
  <Param name="customer_id" type="string">
    Filter orders by customer ID.
  </Param>
  <Param name="sort" type="string">
    Sort field: `created_at`, `-created_at`, `updated_at`.
  </Param>
</ParamTable>
